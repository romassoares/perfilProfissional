Comecei a aprender algumas tecnologias para desenvolvimento web em 2017 como front-end, utilizando html, css , bootstrap e javascript, em um projeto para IFPI - Angical. Posteriormente, em um outros dois projetos para almoxarifado e gestão de RH, utilizamos bootstrap e javascript, utilizamos componentes para otimizar o processo e ajustes, comecei a ler os códigos dos colegas e tentar entender a lógica utilizada para resolver os problemas.

Em seguida, iniciei vários projetos individualmente para aprender tecnologias back-end, concomitantemente já estava ajudando a equipe em coisas simples voltadas a parte lógica do sistema, sempre estudando e tirando as dúvidas com os colegas.

Após todo esse processo de aprendizado constante iniciei e finalizei um projeto como requisito para conclusão do curso com o mínimo de ajuda.

Atualmente estou me especializando em laravel e node na medida do possível.
